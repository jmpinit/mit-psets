Problem Summary

Goal:

Implement F(A, B, C) = C OR A AND B using nfets and pfets.

Constraints:
  • Use no more than 8 mosfets
  • Only nfets in pulldown circuits
  • Only pfets in pullup circuits
Hint:

"Use six mosfets to implement the complement of F and then use the remaining two mosfets to invert that output."
